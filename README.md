# MBD2 Animal Trap

Minecraft 1.20.1 prototype animal trap with 3d Model and animations created using Multiblocked 2

### Dependencies

- [Multiblocked 2](https://github.com/Low-Drag-MC/Multiblocked2)
- [KubeJS](https://github.com/KubeJS-Mods/KubeJS)
